# 神経衰弱ランキング on ニコ生風

60秒でペアを揃えて得点を稼ぐランキング対応の神経衰弱です。
特殊カードを揃えると以後の加点に効果が乗り、スコアが大きく変化します。

## Runtime
Akashic Engine (JavaScript)

## Features
- ルール説明15秒→プレイ60秒→結果の75秒構成
- 通常カード(1〜12)のペアで加点、特殊カードで効果発動
- 発動済み効果を画面上部に常時表示
- 説明/結果画面は高コントラスト配色で読みやすく改善
- カード内テキストは中央揃え

## Controls
- クリック/タップでカードをめくる

## Rules
- 同じカード(数字/特殊)を2枚揃えると成立
- 通常カード: (数字+数字) を基礎点として、発動済み効果(乗算→加算)を適用して加点
- 特殊カード: 2枚揃えると効果が確定し、以後の加点に常時適用
- 制限時間60秒で終了し、スコアをランキング送信(対応環境のみ)
